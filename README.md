# Auto-Test
Application that automatize graphically Selenium test

Auto Test is an application that simplifies automation with Selenium, allowing you to go from design to execution in minutes.

**Main features**:

**Visual Design**: Graphically record test steps.

**Flexible Management**: Save your processes in separate files to organize different test scenarios.

**Standard Format**: Instructions are saved in JSON format, facilitating technical reading and editing.

**Integration**: Can automatize your release cycles by scheduling tests directly in Jenkins, using the available parameters.

**Immediate Feedback**: Monitor the process in real time from the console and obtain detailed reports in HTML.

Support for Chrome and Firefox.
<hr>
Minimum requirement: Java 17, Windows 10<br>
Build using Selenium 4.43
